Внутри программы за часть слова принимается только обычный прямой апостроф с клавиатуры ('). Типографский (изогнутый) апостроф и любые кавычки не являются частью слова.

Within the program, only the standard straight apostrophe (') from the keyboard is considered part of a word. The typographic (curved) apostrophe and any quotation marks are not considered part of a word.
